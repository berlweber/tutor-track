# Tutor Track

## Description
Tutor Track is a small tutoring management app that helps keep track of tutoring assignments and sessions.  
You can create assignments between tutors and students, set learning goals, define the hourly rate, and record tutoring sessions.

I built this project to practice building a full CRUD web application with Django and to simulate a simple tool that could be used by tutoring programs to manage their sessions.

## Getting Started
Deployed app:  
https://tutor-track-9c7f1aa7eb60.herokuapp.com/

Planning materials:  
https://trello.com/b/rLWE4i8g/tutoring-program-manager-django-crud-app-project

Main features:
- Create, view, update, and delete assignments
- Add tutoring sessions to an assignment
- Track session details like date, duration, and notes
